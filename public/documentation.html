Documentation de l'API

Routes disponibles :

  • GET /catways - Récupérer tous les catways.
  • GET /catways/:id - Récupérer les détails d'un catway spécifique.
  • POST /catways - Créer un nouveau catway.
  • PUT /catways/:id - Mettre à jour un catway par son ID.
  • DELETE /catways/:id - Supprimer un catway par son ID.
  • GET /reservations - Récupérer toutes les réservations.
  • GET /reservations/:id - Récupérer les détails d'une réservation.
  • POST /reservations - Créer une nouvelle réservation.
  • PUT /reservations/:id - Mettre à jour une réservation par son ID.
  • DELETE /reservations/:id - Supprimer une réservation par son ID.
